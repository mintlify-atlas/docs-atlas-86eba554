    Quick start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get Print My Bridge running in minutes
  </p>
  
  <Steps>
    <Step title="Download and install">
      Download the latest release for your platform from [GitHub](https://github.com/NaturalDevCR/print-my-bridge/releases):
      
      - **Windows**: `.msi` installer
      - **macOS**: Universal `.app` bundle
      - **Linux**: `.deb` package
      
      Run the installer and launch the application.
    </Step>
    
    <Step title="Generate an API token">
      Open the Print My Bridge application and click **Generate Token** in the settings panel. Copy the generated token — you'll need it to authenticate API requests.
      
      ```bash
      # Example token (yours will be different)
      KmWv0HH0MW2Ex9VrmpSiDkddSUk2141K
      ```
    </Step>
    
    <Step title="Send your first print job">
      Use the API to send a test print job to your default printer:
      
      ```bash cURL
      curl -X POST http://localhost:8765/api/print \
        -H "x-api-token: YOUR_TOKEN" \
        -H "Content-Type: application/json" \
        -d '{
          "content": "Hello, Print My Bridge!",
          "content_type": "text",
          "copies": 1
        }'
      ```
      
      <Accordion title="Response">
      ```json
      {
        "success": true,
        "message": "Texto enviado a impresora exitosamente",
        "job_id": "text_job_123"
      }
      ```
      </Accordion>
    </Step>
  </Steps>
</div>
